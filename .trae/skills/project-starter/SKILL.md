# 项目启动器

## 功能说明

该skill用于启动项目的前端和后端服务：
- 前端服务：使用 `pnpm dev` 命令启动
- 后端服务：使用 `npm run dev` 命令启动

## 使用场景

当用户需要启动项目的前端和后端服务时，调用此skill。

## 执行流程

1. 检查当前目录结构，确定前端和后端目录
2. 分别在前端目录执行 `pnpm dev`
3. 在后端目录执行 `node app.js`

## 示例使用

当用户说："启动项目" 或 "启动前端和后端服务" 时，调用此skill。

## 注意事项

- 确保前端目录中存在 `pnpm` 配置文件和 `dev` 脚本
- 确保后端目录中存在 `app.js` 文件
- 确保已安装必要的依赖
