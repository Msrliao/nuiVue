# 车型配件查询规则

## 规则描述
根据用户输入的车型和配件名称，查询数据库中的库存和价格信息。

## 关键词
车型配件查询, 库存查询, 价格查询, 配件搜索, 快速查询, 模糊匹配, 多条件查询

## 支持的输入格式
1. 完整格式："查询[车型]的[配件]库存"
2. 简化格式："[车型] [配件] 库存"
3. 模糊格式："[车型关键词] [配件关键词]"

## 输出格式
返回结构化的库存和价格信息，包括：
- 车型名称
- 配件名称
- 库存数量
- 单价
- 总价值
- 仓库位置

## 实现逻辑
1. 从用户输入中提取车型和配件名称
2. 调用后端API进行数据库查询
3. 组合查询结果
4. 返回友好的响应信息

## 示例
### 输入
"查询丰田凯美瑞的机油滤清器库存"

### 输出
```
车型：丰田凯美瑞
配件：机油滤清器
库存数量：50个
单价：89元
总价值：4450元
仓库位置：A区1号货架
```

### 输入
"本田CR-V空气滤芯"

### 输出
```
车型：本田CR-V
配件：空气滤芯
库存数量：35个
单价：65元
总价值：2275元
仓库位置：B区3号货架
```
