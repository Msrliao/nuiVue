## 实现右键点击编辑功能

### 修改文件列表

#### 1. base_part/table.vue
- 移除 emitter 导入
- 添加 `edit` 事件到 `defineEmits`
- 修改 `handleSelect` 函数，当选择编辑时触发 `emit('edit', currentRow.value)`

#### 2. base_part/index.vue
- 添加 `editData` 响应式变量存储编辑数据
- 在 TableVue 组件上添加 `@edit` 事件监听器
- 创建 `handleEdit` 函数处理编辑事件，设置编辑数据并打开模态框

#### 3. base_part/addInfor.vue
- 添加 `editData` prop 接收编辑数据
- 添加 `watch` 监听 `editData` 变化，填充表单数据
- 修改 `handleValidateClick` 函数：
  - 检查 `formValue.value.id` 是否存在
  - 存在：调用 `apiClient.put('/part-info/' + id, formValue.value)`
  - 不存在：调用 `apiClient.post('/part-info', formValue.value)`
- 修改 `handleClearForm` 函数，确保正确清空编辑状态

### 优势

1. **统一的编辑界面** - 复用现有的 addInfor.vue 组件
2. **清晰的数据流** - 使用 Vue 标准的 props 和事件
3. **类型安全** - TypeScript 正确推断数据类型
4. **RESTful API 设计** - 使用 POST 新增，PUT 修改
5. **用户体验** - 右键点击直接编辑，操作流畅
